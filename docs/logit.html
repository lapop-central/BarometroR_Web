Modelo de regresi√≥n log√≠stica

Para evaluar los factores asociados al uso de redes sociales se puede calcular un modelo de regresi√≥n. En este caso, a diferencia del modelo de regresi√≥n lineal, la variable dependiente no es num√©rica, sino que se trata de una variable dicot√≥mica, con valores 0, para indicar a los no usuarios, y 1 para identificar a los usuarios de redes sociales.

Con este tipo de variable dependiente no es apropiado usar un modelo de regresi√≥n lineal. Las razones para esto son m√∫ltiples. Un modelo lineal para aproximar la relaci√≥n entre una variable dependiente dicot√≥mica y una independiente num√©rica podr√≠a dar valores predichos mayores que 1 o menores que 0, por lo que no es una buena aproximaci√≥n para calcular probabilidades.

Por el contrario, un modelo de regresi√≥n log√≠stica restringe los resultados entre 0 y 1, por lo que se puede interpretar como una probabilidad. En nuestro ejemplo, se calcula la probabilidad de ser usuario de redes sociales para diferentes valores de las variables independientes.

Otras razones para no usar un modelo de regresi√≥n lineal son m√°s t√©cnicas y tienen que ver con los residuos. Si se usara un modelo de regresi√≥n lineal para aproximar la relaci√≥n entre una variable dependiente dicot√≥mica y una independiente num√©rica se tendr√≠a residuos no normalmente distribuidos y hereroced√°sticos

El modelo se calcula con el comando glm donde se indica la variable Y y luego las variables independientes. Cada variable independiente se suma al modelo. El modelo incluye efectos fijos por pa√≠s. En la secci√≥n sobre regresi√≥n lineal m√∫ltiple se explic√≥ el uso de efectos fijos por pa√≠s para una regresi√≥n lineal, es decir, cuando la variable dependiente es num√©rica. Aqu√≠ se usar√° el mismo comando factor() para incluir variables dummy de cada pa√≠s, tomando a M√©xico (pa√≠s 1) como referencia.

Este modelo de guarda en un objeto ‚Äúmodelo1‚Äù el que se puede describir con el comando summary.
